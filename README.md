# Reise-Quiz 🌍

Interaktive Quiz-Spiele rund um meine Reiseziele. Wähle auf der Startseite ein
Land und stelle dein Wissen in der ultimativen Herausforderung unter Beweis.

## Quizze

| Quiz | Datei |
|------|-------|
| 🇹🇭 Thailand | [`thailand.html`](thailand.html) |
| 🇱🇰 Sri Lanka | [`sri-lanka.html`](sri-lanka.html) |

## Lokal starten

Es sind reine HTML-Dateien ohne Build-Schritt – einfach `index.html` im Browser öffnen:

```bash
open index.html        # macOS
```

## Struktur

```
index.html       → Auswahl-Menü (Startseite)
thailand.html    → Thailand-Quiz
sri-lanka.html   → Sri-Lanka-Quiz
```
